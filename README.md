# Hi!
Code chính sẽ ở trong folder **VirtualMouse**
chạy dự án bằng python vm_gui.py
## ===== Lưu ý =====
Chạy bằng **VSCode ở trong Anaconda** và bắt buộc dùng **Python 3.8**
Nhất định phải là python 3.8 nha

## Cách chạy dự án
Chỉ cần copy chạy lần lượt các lệnh sau chạy ở terminal:

conda create -n virtualmouse python=3.8
conda activate virtualmouse
pip install -r requirements.txt

## Lưu ý 2
- cái tên virtualmouse là tên môi trường ảo, nếu trùng tên hoặc có rồi thì kiểm tra xem có đang chạy trên python 3.8 với mediapipe 0.10.7 hay không hoặc đặt tên khác để chạy hoặc xóa đi cài theo Đức
- mediapipe phải là phiên bản 0.10.7
- nếu terminal là cmd thì không được chạy code bằng hình tam giác trên vscode vì hấn sẽ nhảy qua môi trường bình thường
- mấy file bên trong thư mục mediapipe7 là nơi để học code, nên code lại cho nhớ, nếu code được đến bài 8 là coi như hoàn thành bài tập lớn rồi
- cách chạy file trên terminal: python tênfile.py
